FireDragon TOM Modfix (Portable)

1) Put broken mods into the Source folder.
2) Run Setup.bat to verify package files and tool availability.
3) Run Launch.bat to process mods.
4) Collect fixed mods from the Output folder.
5) Check run logs in the Logs folder (one main log + one detailed log per program).

Configuration:
- Edit config.json to change Source/Output/program paths and update behavior.

- Optional: set flatten_script in config.json to control post-processing flatten step.
